# PRODIGY_WD_02
A responsive and interactive stopwatch web app built with HTML, CSS, and JavaScript, featuring start, pause, reset, and lap tracking functionality.

🚀 Features
  >  Precision Timing: Accurately tracks seconds, minutes, and hours (or milliseconds).
  >  Lap/Split System: Allows users to record specific time intervals without stopping the main timer.
  >  Responsive Design: Optimized for both desktop and mobile screens.
  >  Intuitive UI: Clean and modern interface with instant button responsiveness.
